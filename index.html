Google Sheets 연동

Google Sheets를 DB로 사용합니다. 담당자가 시트에 데이터를 입력하면 대시보드에 자동으로 반영돼요.

📋 시트 준비 방법 (처음 한 번만)
① Google Sheets에서 세미나 시트와 참여자 시트를 만들어요
② 공유 → "링크가 있는 모든 사용자 → 뷰어" 로 설정
③ 아래에 각 시트의 URL을 붙여넣고 저장
📅 세미나 시트 헤더 (1행):
세미나명 · 날짜 · 종류 · 신청자수 · 최대참석자 · 참석률 · 강사 / 학원 · 영상 · 교안 · 메모
👤 참여자 시트 헤더 (1행) — 1행 = 세미나 참여 1건:
날짜 · 회원ID · 이름 · 연락처 · 세미나명
중복 참여 시 여러 행으로 입력 → 자동으로 참여 횟수 집계
세미나 시트 URL (탭별로 여러 개 추가 가능)
